# Mobile MVP Outline

## Positioning

`Baby Weekly Companion` is a mobile app for English-speaking new parents.

It is not:

- a generic baby tracker
- a parenting content feed
- an AI doctor
- a free-form chat toy

It is:

- a proactive baby planning agent
- a weekly action organizer
- a calm coordination layer for parents and caregivers

## Core screens

### 1. Today

Purpose:
- show the 2 to 4 most important actions right now
- reduce decision fatigue

Key modules:
- urgent reminders
- age-stage guidance
- weather-aware adjustments

### 2. Planner

Purpose:
- turn milestones and appointments into a weekly rhythm

Key modules:
- weekly priorities
- appointment preparation
- next best action

### 3. Outings

Purpose:
- help parents leave home with less chaos

Key modules:
- context-aware wear guidance
- packing checklist
- outing duration suggestions

### 4. Family

Purpose:
- keep all caregivers aligned

Key modules:
- shared task ownership
- reminder sync
- handoff clarity

## V1 feature boundaries

Include:

- baby profile
- age-based weekly planning
- appointment reminders
- outing checklists
- caregiver coordination

Do not include yet:

- open-ended AI chat
- diagnosis
- medical decision engine
- deep analytics
- large content library

## Monetization direction

Free:

- one baby profile
- basic weekly plan
- simple reminders

Paid:

- family sharing
- advanced summaries
- AI-generated checklists
- premium planning flows
- multi-child management
