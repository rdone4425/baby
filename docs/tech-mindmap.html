Tech Mindmap

Baby Planning Agent Tech

技术上不应该先做一个万能 AI，而应该搭一个规则优先、状态驱动、半主动触发的 agent system：移动端负责承载体验，rules engine 决定骨架，LLM 负责解释、整理和输出。

关键原则：规则做骨架，AI 做肌肉和皮肤；高风险医疗判断一律不交给模型。

整体架构

  • Mobile App
  • Agent Service
  • Rules Engine
  • Data Store
  • Notification Service
  • External APIs

Mobile App

  • React Native / Expo
  • Today 页面

Agent Service

  • 状态感知
